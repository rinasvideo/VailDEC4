<h1>VailDEC4</h1>
<p>このソフトウェアは簡単に使用できることを目標に制作された操作が簡単なファイル暗号化ソフトウェアです。</p>
<p>フリーソフトウェアとして配布されています。</p>
インストーラーなどのアーカイブも、Releseで配布してありますが、 以下の公式サイトでも配布しております。そちらかもどうぞ。
